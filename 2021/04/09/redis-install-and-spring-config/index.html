设置密码

redis密码设置有两种方式，一种需要重启redis服务，一种不需要重启redis服务。

首先，介绍一下需要重启redis服务的设置方式 即找到redis的配置文件—redis.conf文件，然后修改里面的requirepass，这个本来是注释起来了的，将注释去掉，并将后面对应的字段设置成自己想要的密码，保存退出。重启redis服务，即可。
